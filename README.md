# Little Lemon Booking App

This is the Meta Front-End Developer Capstone project: Booking a table on the Little Lemon website.

## Demo
[Live Demo](https://el-ahmed.github.io/little-lemon/)

## Features
- Booking form with validation and error messages
- Accessible form fields with labels
- Responsive design
- React Router for navigation
- Unit tests with Jest

## Tech Stack
React, React Router DOM, HTML, CSS, JavaScript

## Run Locally

```bash
git clone https://github.com/El-Ahmed/little-lemon.git
cd little-lemon
npm install
npm start
